Listen Toolbar

The Listen toolbar is displayed across the top of the application window, below the toolbar. The Listen toolbar contains the Local Port, Remote Port and URL fields and Listen button.

Example 1.

You want use Port Sniffer for connect to bonalink.com site from your local machine. Please, type the following values for it:

  • Local Port: 80
  • Remote Port: 80
  • URL: bonalink.com

Press the Listen button and navigate to localhost from your web browser.

Example 2.

You want use Port Sniffer for connect to your local web server from your local machine. Please, type the following values for it:

  • Local Port: 81
  • Remote Port: 80
  • URL: localhost

Press the Listen button and navigate to localhost:81 from your web browser.

Example 3.

You want use Port Sniffer for connect to bonalink.com site from web bowser on another machine. Name of your local machine is my_machine. Please type the following values for it:

  • Local Port: 80
  • Remote Port: 80
  • URL: bonalink.com

Press the Listen button and navigate to my_machine from your web browser on another machine.

Example 4.

Sometimes you want use Port Sniffer for connect to two or more sites. For example, you want connect to bonalink.com site and your local web server. Launch two or more instance of Port Sniffer for it:

For connect to bonalink.com site, please type the following values into first instance of Port Sniffer:

  • Local Port: 81
  • Remote Port: 80
  • URL: bonalink.com

Press the Listen button and navigate to localhost:81 from your web browser.

For connect to your local web server, please type the following values into second instance of Port Sniffer:

  • Local Port: 82
  • Remote Port: 80
  • URL: localhost

Press the Listen button and navigate to localhost:82 from your web browser.
