Port Sniffer Help Index

My utility can be useful for web developers and other programmers who are used sockets in his programs. You can use my utility for spying on socket's traffik. For example you can see content of the data exchange between your web browser and web server or your e-mail client and SMTP server.

How it works?

My utility is a bridge between your client and server. Your client connects to my utility instead of the server. My utility connects to the server instead your client. Port Sniffer listens an incoming connection for selected port on your local machine. After accepting of connection, Port Scanner tries to connect to the selected port of the server and relays all data from accepted connection to the server. Port Sniffer then waits a reply from the server and relays it to the incoming connection. You can see all relayed data on the main window.

How to use the Port Sniffer?

  • Please type a port number, URL of the server and press the Listen button. Now my utility listens for an incoming connection. Service Name and Transport Protocol Port Number Registry.
  • Prepare your client (a web browser, e-mail client or something like that). You need to replace in your client the URL of the server to URL of the computer, where Port Sniffer is listens for connection (usually to localhost).
    • For example you can see a data exchange between your web browser and web server. For it:
      • Type 80 in the Port: field, "bonalink.com" in the URL: field and press the Listen button. Port Sniffer listens for an incoming connection.
      • Open a web browser and navigate to http://localhost .
    • In another example you can see a process of sending an e-mail to the SMTP server.
      • Type 25 in the Port: field, "smtp.mail.ru" in the URL: field and press the Listen button. Port Sniffer listens for sending an e-mail.
      • Open an e-mail client, for example Outlook Express.
      • Go to Tools / Accounts... menu item. The Internet Accounts dialog opens.
      • Go to Mail tab and open the properties of your e-mail account. The Properties of your e-mail account dialog opens.
      • Go to Servers tab and type "localhost" in the Outgoing mail (SMPT): field.
      • Press OK and close all dialogs.
      • Try to send an e-mail. Now you can see all data exchange between your e-mail client and SMTP server in the main window of the Port Scanner.
